# AUTO-UPB

This repository was created for an university project about Networks, Project Gerency and Database Administration subjects. 

Wearcar is a company that sells vehicle accessories, belonging to the self-named AUTO-UPB consortium. **The above names are fictitious and are used just for academic purposes.**
